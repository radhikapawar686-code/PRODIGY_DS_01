# Task 1: Visualizing Population Data Using Bar Charts and Histograms in Python

## Overview
This project demonstrates how to visualize categorical and continuous data using Python. The dataset contains information about individuals' ages and genders. Two types of visualizations are created:

- **Bar Chart**: Displays the distribution of genders in the dataset.
- **Histogram**: Displays the distribution of ages in the dataset.

These visualizations help in understanding data patterns and are commonly used in Exploratory Data Analysis (EDA).

## Objectives
- Learn how to create a bar chart for categorical variables.
- Learn how to create a histogram for continuous variables.
- Understand the distribution of age and gender data.
- Gain hands-on experience with Python data visualization libraries.

## Dataset
The sample dataset includes:
- **Age** (Continuous Variable)
- **Gender** (Categorical Variable)

Example:

| Age | Gender |
|------|--------|
| 18 | Female |
| 19 | Male |
| 20 | Female |
| 21 | Male |
| 22 | Female |
